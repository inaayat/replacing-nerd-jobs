# Seattle Open Budget

Interactive dashboard showing approved operating budget spend per program, 2018–2026.

## Deploy to Vercel

**Option A — Vercel CLI**
```bash
npm i -g vercel
vercel
```

**Option B — Vercel Dashboard**
1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new) and import the repo
3. Leave all build settings at their defaults and click **Deploy**

No build step or dependencies required — Vercel serves `index.html` directly.

## Data source

Seattle Open Budget (`operating_budget-2026-05-31.csv`), exported 2026-05-31.
